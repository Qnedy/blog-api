# Blog API

Simple REST API to study Fastify.

[x] The user must be able to list posts

[x] The user must be able to create a post
	- username
	- post title
	- post content

[x] The user must be able to comment in a post
	- username
	- comment content

[x] The user must be able to like a post

[x] The user must be able to delete a post
